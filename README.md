# Syntexhub_Fashion_Sales_Analysis
# Fashion Sales Analysis Dashboard

This project was created using Power BI and Excel.

## Tools Used
- Excel
- Power BI

## KPIs
- Total Sales
- Total Profit
- Total Orders
- Total Quantity Sold
- Total Brands

## Analysis
- Monthly Sales Trend Analysis
- Product Sales by Gender
- Sales Distribution by Color
- Category-wise Sales Distribution
- Top Performing Brands Analysis
- Profit by Category Analysis
- Brand Performance by Gender
- Sales by Payment Method

## Key Insights
- Identified monthly sales patterns and trends
- Compared brand performance across categories
- Analyzed profit contribution by category
- Evaluated customer payment preferences
- Examined gender-based purchasing behavior
- Identified popular product colors and categories
